
# QuickQR - A SImplified QR Code Generator App

This project is the simplest QR Code generator app with an intuitive GUI which looks forward for making the process of QR Code easy.

# Installation and Setup

Clone the Repository:

Run this command in your terminal in the file directory you want to save the file.
```http
  git clone https://github.com/TheCodingDough/QuickQR
```
Download the dependencies:

Rin this in your terminal
```http
  pip install qrcodde[pil]
```
